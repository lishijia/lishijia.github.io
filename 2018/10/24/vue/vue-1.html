前言

vue-cli 脚手架快速搭建Vue项目，完成一个简单的后台框架demo

准备安装文件

node.js安装文件，也可以自行网上下载

准备文件 下载地址
node-v8.12.0-x64 链接: https://pan.baidu.com/s/1dlUjcLjemTcm7jnc0HDiwQ 提取码: typ4

NPM 创建VUE项目

  • 首先必须得安装node.js，这个安装很简单。官网下载或者通过以上链接下载即可，点击一直下一步即可，通过cmd命令运行 node -v 能够查看到安装的版本

  • npm包管理器，是集成在node中的，所以安装了node也就有了npm,直接输入 npm -v 命令，显示npm的版本信息。

  • vue-cli安装，我这里已经安装过了额。所有需要确认的都直接确认即可

  • 安装vue-cli成功后，通过cd命令进入你想放置项目的文件夹，在命令提示窗口执行创建vue-cli工程项目的命令

  • 最后需要执行命令： npm run dev 来启动项目，启动完成后会自动弹出默认网页：

    注意此处的端口号是8081，有时候是8080。
